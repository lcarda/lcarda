- 🤖 Machine Learning Engineer with experience in real-time pipelines, anomaly detection, computer vision, and RAG systems

## Current work

My academic work focuses on **KZG-based counting and non-membership proofs**, using algebraic methods, polynomial interpolation, and bilinear pairings to build succinct and efficient proof systems.

Alongside that, I work on machine learning engineering problems such as:

- real-time video processing pipelines
- object detection with YOLO
- anomaly detection with autoencoders
- RAG systems with LangChain and LangGraph
- evaluation and design of prompts for LLM-based workflows
